# Roblox One 2.1.4

Roblox One 2.1.4 is the first packaged Windows release.

## Highlights

- Keep any number of Roblox website sign-ins in isolated local account
  profiles, with custom labels and colors.
- Remember a destination for each account while sharing Recent and Favorites
  across the app.
- Launch public places, private-server links or codes, and supported server IDs
  recovered from recent launches.
- Pin and rename saved experiences without losing their canonical destination.
- Use a sequential batch launch with practical delay choices, or close every
  visible and background Roblox Player process from one control.
- Choose subtle interface sounds and one of the included startup sounds, or
  import a local sound of your own.
- Review release notes and confirm before downloading or installing an update.

## Safety and compatibility

Account data stays under `%LOCALAPPDATA%\RobloxOne`; no account profiles or
credentials are included in the installer. Roblox web requests use official
Roblox endpoints, and Roblox Player is signature-checked before launch.

Roblox ultimately controls whether multiple Player processes can run, so batch
launching remains best-effort. The optional HandleScope and generic launch-hook
connectors are local-only, disabled by default, and not required for ordinary
use.

Before closing existing clients, batch mode now verifies every selected account
and supported private-server share code. A running batch can be cancelled, and
the account selected before the batch is restored afterward. Optional local
launch integrations finish (or safely skip) before the app reports their step
complete, without changing whether Roblox itself launched successfully.

Settings writes keep a validated local backup. If the primary file becomes
unreadable, Roblox One preserves it and recovers the backup; if neither file can
be read, browser profiles are left untouched and automatic cleanup stays paused
across restarts. Interrupted new-account profiles are reconciled on the next
reliable startup. Destination errors are shown next
to the field, and history clearing follows both the public/private and account
filters while always preserving Favorites. Launches use the selected Roblox
account's official locale when available instead of always forcing English.

Release packaging is fail-closed: dependencies and the self-contained runtime
are pinned, packages carry complete upstream notices, the signed release
descriptor binds the exact package hash and metadata, and the exact artifact
inventory is covered by an SPDX SBOM, SHA-256 checksums, and GitHub
attestations. Windows executables are not Authenticode code-signed, so Windows
may report an unknown publisher. The project is distributed under the MIT
License.
