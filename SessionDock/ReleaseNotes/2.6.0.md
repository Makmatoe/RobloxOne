SessionDock 2.6.0

One-click HandleScope setup

- The HandleScope integration now includes an Install Latest HandleScope release button. SessionDock downloads the current stable Windows x64 release, verifies its published hashes and internal file inventory, and installs it for the current Windows user.
- A successful install starts the HandleScope API immediately and enables its limited per-user autostart at Windows sign-in. After the one-time SessionDock integration opt-in, users no longer need to start HandleScope again after each sign-in.
- Installation never requests elevation. HandleScope remains optional, is not bundled with SessionDock, and is rejected before execution if its release metadata, download, archive layout, or internal manifest fails validation.

HandleScope is not Authenticode-signed. SessionDock verifies that the package matches the canonical immutable GitHub release, but Windows may still identify its publisher as unknown.
