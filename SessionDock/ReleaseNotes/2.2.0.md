# SessionDock 2.2.0

SessionDock now completes the application rename introduced by the 2.1.5
compatibility bridge.

## Canonical SessionDock release

- The application executable, installer, portable archive, update descriptor,
  SBOM, release channel, and public documentation now use the SessionDock name.
- Updates come from the protected `Makmatoe/SessionDock` repository.
- Existing installed copies continue updating in place through Velopack's
  retained private package identity.
- Existing local accounts, isolated browser sessions, labels, colors, history,
  favorites, sounds, and HandleScope opt-in remain under
  `%LOCALAPPDATA%\SessionDock`.

## Destination controls

- **Set for all** copies the currently validated destination to every saved
  account in one action.
- Tracked Server Job IDs show an inline warning that Roblox may close the Player
  if the selected server is no longer available.
- Validation occurs before any account destination is changed.

This release remains local-first, contains no account data, bundles no
HandleScope software, and adds no third-party dependencies.
