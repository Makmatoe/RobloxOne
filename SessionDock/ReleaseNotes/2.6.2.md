SessionDock 2.6.2

Security-hardened releases and integrations

- Updated the self-contained Windows runtime to .NET 10.0.10 and the build SDK to 10.0.302, with automatic checks for future security patches.
- Strengthened Roblox executable signature validation with online certificate revocation checks and fresh validation before sensitive actions.
- Added independent signed-release and installed-file verification for HandleScope, including stricter archive and download validation.
- Made production releases require timestamped Authenticode signatures and managed OIDC-backed descriptor signing with no unsigned fallback.
- Removed the isolated smoke-test path from production builds and strengthened dependency, workflow, and repository policy checks.

HandleScope installation remains unavailable in builds that do not include its genuine production release public key. Production packaging also stops clearly until the protected managed-signing variables are configured.
