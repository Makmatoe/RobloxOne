SessionDock 2.5.0

Join by user and safer isolated sessions

- Join a Roblox user by username, user ID, or profile link when their privacy settings and current session allow it. SessionDock resolves the user's current server and still requests a fresh launch ticket for your selected account.
- A missing, damaged, or unregistered Microsoft Edge WebView2 runtime no longer closes SessionDock while an isolated account session is loading. The app keeps accounts and browser profiles unchanged, returns to the launcher, and offers the official Microsoft repair and download page.
- The exact WebView2 failure reported by Windows as 0x80004004 is now contained, while unrelated COM and programming errors remain visible to diagnostics.

This release adds no telemetry and sends join lookups only to official Roblox endpoints.
