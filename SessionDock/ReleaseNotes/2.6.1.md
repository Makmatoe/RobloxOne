SessionDock 2.6.1

Simpler SessionDock installation

- Every repository README now offers one Install Latest SessionDock release button that downloads the correct Windows x64 Setup directly from the current stable release.
- The installation guide now explains the short first-time setup path and keeps checksum verification available for users who want an independent download check.
- Maintainer validation now uses the repository's complete build command, and release packaging reads the exactly pinned Velopack version from the checked-in tool manifest.

The installer remains a standard-user, per-user installation. Windows may show Unknown publisher because the project uses a no-cost release-verification model instead of a paid Authenticode certificate.
