# SessionDock 2.1.5

This is the compatibility update that renames Roblox One to SessionDock while
keeping existing installations, accounts, browser sessions, favorites, and
history usable.

## Destination controls

- A new **Set for all** button copies the currently validated destination to
  every saved account.
- A tracked Server Job ID now shows a persistent inline warning that Roblox may
  close the Player if that server can no longer be found.
- Invalid destinations are rejected before any account is changed.

## SessionDock migration

- The application UI, project structure, documentation, and diagnostics now use
  the SessionDock name.
- Existing local data is moved from `%LOCALAPPDATA%\RobloxOne` to
  `%LOCALAPPDATA%\SessionDock` before the renamed app uses it.
- The old single-instance identity is retained privately during migration so an
  older copy cannot open the same browser profiles at the same time.
- The legacy package identity remains in this one bridge release so installed
  2.1.4 copies can verify and apply it safely.

No account credentials, browser data, or HandleScope binaries are added to the
release package.
